 Программа: Деление целых чисел с приведением типов
 * Описание: Программа демонстрирует важную особенность языка Си - 
 *           деление целых чисел и приведение типов (type casting)
 *           для получения дробного результата
 * 
 * Задача: Разделить целое число a на целое число b и получить
 *         дробный результат с одним знаком после запятой
 * 
 * Математически: 5 / 2 = 2.5
 * 
 * Проблема: Если просто написать a / b, то в Си получится 2, а не 2.5!
 *           Потому что при делении двух целых чисел результат тоже целый
 * 
 * Решение: Использовать приведение типов (float) для получения дробного результата
 * 
 * Пример вывода:
 * a / b = 2.5
 * 
 * Используемые технологии:
 * - Язык программирования: C (стандарт C99/C11)
 * - Библиотека: stdio.h (для ввода/вывода)
 * - Операция: приведение типов (type casting) - (float)
 * - Формат вывода: %.1f (один знак после запятой)
 * - Компилятор: GCC / MinGW / Clang / любой другой компилятор Си
 * 
 * Инструкция по запуску:
 * 1. Сохранить файл как division.c
 * 2. Компиляция: gcc division.c -o division
 * 3. Запуск: 
 *    - Linux/macOS: ./division
 *    - Windows: division.exe
 * 4. Или использовать онлайн-компилятор (onlinegdb.com, replit.com)

  * Программа: Проблема точности вычислений с плавающей точкой
 * Описание: Программа демонстрирует важную особенность компьютеров -
 *           неточность представления дробных чисел в двоичной системе
 * 
 * Математически: 0.1 + 0.2 = 0.3
 * В компьютере:  0.1 + 0.2 ≈ 0.30000000000000004441 (или другое близкое значение)
 * 
 * Это происходит потому, что числа 0.1 и 0.2 в двоичной системе 
 * имеют бесконечную периодическую дробь, как 1/3 в десятичной
 * 
 * Пример вывода (может отличаться на разных системах):
 * 0.30000001192092895508
 * 
 * Используемые технологии:
 * - Язык программирования: C (стандарт C99/C11)
 * - Библиотека: stdio.h (для ввода/вывода)
 * - Тип: float (числа с плавающей точкой одинарной точности)
 * - Суффикс: f (явно указывает, что число типа float)
 * - Формат: %.20f (вывод с 20 знаками после запятой)
 * - Стандарт: IEEE 754 (стандарт представления дробных чисел в компьютерах)
 * 
 * Инструкция по запуску:
 * 1. Сохранить файл как float_precision.c
 * 2. Компиляция: gcc float_precision.c -o float_precision
 * 3. Запуск: 
 *    - Linux/macOS: ./float_precision
 *    - Windows: float_precision.exe
 * 4. Или использовать онлайн-компилятор (onlinegdb.com, replit.com)

 * Программа: Целочисленное переполнение (integer overflow)
 * Описание: Программа демонстрирует важнейшее понятие в программировании -
 *           переполнение целочисленного типа. Когда число выходит за пределы
 *           допустимого диапазона для данного типа, происходит "заворот"
 * 
 * Математически: 2147483647 + 1 = 2147483648
 * В компьютере (для 32-битного int): 2147483647 + 1 = -2147483648
 * 
 * Это происходит потому, что 2147483647 (0x7FFFFFFF) - максимальное 
 * положительное число для 32-битного int со знаком. При добавлении 1
 * биты переполняются и число становится минимальным отрицательным
 * 
 * Пример вывода:
 * Было: 2147483647
 * Стало: -2147483648
 * 
 * Используемые технологии:
 * - Язык программирования: C (стандарт C99/C11)
 * - Библиотека: stdio.h (для ввода/вывода)
 * - Тип: int (32-битное целое со знаком)
 * - Формат: %d (для вывода целых чисел)
 * - Компилятор: GCC / MinGW / Clang / любой другой компилятор Си
 * 
 * Инструкция по запуску:
 * 1. Сохранить файл как overflow.c
 * 2. Компиляция: gcc overflow.c -o overflow
 * 3. Запуск: 
 *    - Linux/macOS: ./overflow
 *    - Windows: overflow.exe
 * 4. Или использовать онлайн-компилятор (onlinegdb.com, replit.com)
